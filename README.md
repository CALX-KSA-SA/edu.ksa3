## Customization

### To change images:
Edit the background image URLs in `styles.css`:
- `.hero` - Main hero section
- `.why-saudi-section` - Why Saudi Arabia section
- `.differentiators-section` - What sets us apart section
- `.cta-section` - Call to action section

### To modify content:
Edit the `data-en` and `data-ar` attributes in `index.html`

### To adjust colors:
Modify CSS variables in `styles.css` under `:root`
